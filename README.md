Rock, Paper, Scissors Game
A lightweight, interactive, and fun implementation of the classic Rock, Paper, Scissors game played against a computer.

1. Features
Interactive Gameplay: Play against a randomized, fair computer opponent.

Score Tracking: Keeps track of wins, losses, and ties during your session.

Responsive Design: Works smoothly across both desktop and mobile terminals/browsers.

Clean UI: Simple, intuitive controls and instant feedback on who won each round.

2. Built With
Frontend: HTML5, CSS3, JavaScript
